Awesome-emojify

A jQuery plugin to convert Emoji keywords to images

View on GitHub Download .zip Download .tar.gz

Why

I created this plugin, so that in an easy way without any skills you can use emoticons on your own site

Installation

Just add it to your page head: <script scr="awesome-emojify.min.js></script> or (for development) <script scr="awesome-emojify.js></script>
